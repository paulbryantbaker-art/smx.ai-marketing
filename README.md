# smbx.ai Marketing Command Center

Pre-launch LinkedIn campaign for smbx.ai — the AI deal intelligence platform for M&A. This repo is the single source of truth for all marketing strategy, content, brand assets, and execution tracking.

---

## Quick Start

**Start here:** Read [SMBX_MARKETING_MASTER_PLAN.md](SMBX_MARKETING_MASTER_PLAN.md) — it contains the complete strategy, all 20 ready-to-post texts for Weeks 1-4, carousel outlines, avatar scripts, competitive positioning, branded terms, and the execution system.

## File Map

| File / Folder | What It Does |

| `SMBX_MARKETING_MASTER_PLAN.md` | **The single source of truth.** Complete LinkedIn GTM strategy, content calendar, all 20 post drafts (Weeks 1-4), carousel outlines, HeyGen scripts, branded terms, competitive positioning, and execution system. |
